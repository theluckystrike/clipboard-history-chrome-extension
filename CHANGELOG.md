# Changelog

All notable changes to Clipboard History will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2024-12-01

### Added
- Automatic clipboard capture on copy events
- IndexedDB storage for up to 10,000 clipboard items
- Instant search across all clipboard history
- Star/favorite clips for quick access
- Pin important clips to prevent deletion
- Smart content type detection (text, URL, email, code, JSON, number)
- One-click copy from history
- Direct paste into editable fields via content script
- Context menu integration ("Paste from Clipboard History")
- Keyboard shortcut (Cmd+Shift+V / Ctrl+Shift+V)
- Filter by type: All, Starred, Pinned, Text, URL, Code
- Export clipboard history as JSON
- Dark theme with light theme option
- Settings panel (notifications, duplicates, theme)
- Welcome onboarding flow
- Internationalization support (English, Spanish, Japanese)
- Accessibility features (focus rings, keyboard navigation, reduced motion)
- Duplicate detection (skips consecutive identical copies)
